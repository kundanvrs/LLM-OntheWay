

# 📦 LangSmith Demo

A simple project demonstrating how to use **LangSmith** for tracing, monitoring, and improving LLM-based applications, along with a beginner-friendly explanation of DevOps concepts.

---

## 🚀 Installation

```bash
uv add langsmith
```



## 🧠 What is LangSmith?

LangSmith is a platform that helps developers:

* Debug LLM applications
* Trace requests and responses
* Monitor performance
* Improve prompt quality

---

## 🌐 Access Dashboard

👉 [https://smith.langchain.com/](https://smith.langchain.com/)

## 📊 Example Output

```text
DevOps is like a teamwork between developers (who build software) and IT operations (who manage and maintain systems). The goal is to make the whole process faster and smoother by using tools and practices that help both teams work together efficiently.

Here's how it works:
- Automation: Tools like CI/CD pipelines automatically test and deploy code, reducing errors.
- Collaboration: Developers and ops work together from idea to launch, sharing responsibilities.
- Continuous: Software is tested and released constantly, so problems are caught early.

Think of it as a "superpower" for teams: it helps build better software faster, with fewer mistakes. 🚀
```
